# Contributing to EntPTC

Thank you for your interest in the EntPTC project.

## Project Status

This repository contains the implementation and validation code for the EntPTC consciousness model. The codebase is provided for reproducibility and academic reference.

## How to Contribute

### Reporting Issues

If you find bugs or have questions about the implementation:

1. Check existing issues to avoid duplicates
2. Open a new issue with a clear description
3. Include relevant error messages and system information
4. Provide steps to reproduce the problem

### Code Contributions

For code contributions:

1. Fork the repository
2. Create a feature branch from `main`
3. Make your changes with clear commit messages
4. Ensure code follows existing style conventions
5. Submit a pull request with a description of changes

### Documentation

Improvements to documentation are welcome:

1. Fix typos or clarify explanations
2. Add examples or usage instructions
3. Update outdated information

## Code Style

- Python code follows PEP 8 conventions
- Use descriptive variable and function names
- Include docstrings for functions and classes
- Keep functions focused and modular

## Testing

Before submitting changes:

1. Verify code runs without errors
2. Check that existing functionality is preserved
3. Test with sample data if applicable
