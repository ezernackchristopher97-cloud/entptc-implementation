### 4. Rotor Construction: Π(q) = e^(-B/2)

**ENTPC.tex:** "Mapping Π: H_H → Cl(3,0) via rotor Π(q) = e^(-B/2)"

**Mathematical Formula:**
```
Rotor: R = e^(θB/2) = cos(θ/2) + sin(θ/2)B
```
where B is a unit bivector.

**EntPTC Implementation (clifford.py, lines 193-234):**
```python
def rotor_from_bivector(bivector: CliffordElement, angle: float) -> CliffordElement:
 """
 Create rotor from bivector.
 
 Per ENTPC.tex: Π(q) = e^(-B/2) where B is bivector.
 
 Rotor: R = e^(θB/2) = cos(θ/2) + sin(θ/2)B
 """
 # Extract bivector components
 B = bivector.grade_projection(2)
 
 # Compute norm of bivector
 B_norm = np.sqrt(B.e12**2 + B.e23**2 + B.e31**2)
 
 # Normalize bivector
 B_unit = CliffordElement(
 e12=B.e12/B_norm,
 e23=B.e23/B_norm,
 e31=B.e31/B_norm
 )
 
 # Rotor: cos(θ/2) + sin(θ/2)B_unit
 half_angle = angle / 2.0
 rotor = CliffordElement(
 scalar=np.cos(half_angle),
 e12=np.sin(half_angle) * B_unit.e12,
 e23=np.sin(half_angle) * B_unit.e23,
 e31=np.sin(half_angle) * B_unit.e31
 )
 
 return rotor
```

**✅ Verification:**
- Correct exponential formula: e^(θB/2) = cos(θ/2) + sin(θ/2)B
- Normalizes bivector before constructing rotor
- Returns even element (scalar + bivectors)
- Matches ENTPC.tex specification exactly

**Mathematical Proof:**
- For unit bivector B² = -1
- Taylor series: e^(θB/2) = Σ (θB/2)^n / n!
- Separating even/odd terms: cos(θ/2) + sin(θ/2)B ✓
