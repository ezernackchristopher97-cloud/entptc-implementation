## Key Features of This Schematic

1. **Unidirectional flow**: Geometry → Operator → Collapse → Invariants → Projection → (optional) Description
2. **Causal labels**: Each stage clearly marked as PRIMARY, CAUSAL, CONTROL, etc.
3. **Forbidden paths**: Explicit violations shown in red box
4. **Absurdity Gap**: Positioned as post-operator diagnostic, not in main flow
5. **No wave language**: "Timescale" and "dimensionless rate" instead of "frequency"
