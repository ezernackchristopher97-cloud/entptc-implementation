# Archive Directory

This directory contains archived materials that are not part of the primary repository but are preserved for reference.

## Contents

- **tex_versions/**: Previous versions of the LaTeX manuscript
- **bib_versions/**: Previous versions of the bibliography file
- **pdf_versions/**: Previous compiled PDF versions
- **tests/**: Test and debug scripts
- **logs/**: Processing and compilation logs
- **misc/**: Miscellaneous auxiliary files
- **reference/**: Reference materials
- **enhanced/**: Enhanced algorithm implementations
- **data_archives/**: Data archive pointers

## Note

These files are archived for historical reference. The authoritative versions are in the root directory.
